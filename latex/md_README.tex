This command-\/line utility allows you to encrypt and decrypt files using the Triple DES (3DES) algorithm in CBC mode. The cryptographic key and initialization vector (IV) are derived from a user-\/provided password using Open\+SSL\textquotesingle{}s key derivation functions.\hypertarget{md_README_autotoc_md1}{}\doxysection{Features}\label{md_README_autotoc_md1}

\begin{DoxyItemize}
\item {\bfseries{Encryption}}\+: Encrypt files using the 3DES (Triple DES) algorithm in CBC mode.
\item {\bfseries{Decryption}}\+: Decrypt previously encrypted files.
\item {\bfseries{Password-\/\+Based Key Derivation}}\+: The key and IV are derived securely from the userâ€™s password using SHA-\/256.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md2}{}\doxysection{Prerequisites}\label{md_README_autotoc_md2}

 \hypertarget{md_README_autotoc_md7}{}\doxysection{Usage}\label{md_README_autotoc_md7}
The utility takes several command-\/line arguments to specify whether you want to encrypt or decrypt a file, along with input, output, and password parameters.


\begin{DoxyCode}{0}
\DoxyCodeLine{./des [-\/e|-\/d] -\/i input -\/o output -\/p password}

\end{DoxyCode}


Options\+:


\begin{DoxyItemize}
\item -\/e \+: Encrypt the input file
\item -\/d \+: Decrypt the input file
\item -\/i \+: Path to the input file
\item -\/o \+: Path to the output file
\item -\/p \+: Password to use for encryption or decryption
\end{DoxyItemize}\hypertarget{md_README_autotoc_md8}{}\doxysection{Example Usage}\label{md_README_autotoc_md8}
\hypertarget{md_README_autotoc_md9}{}\doxysubsection{Encrypt a File}\label{md_README_autotoc_md9}
To encrypt a file named plaintext.\+txt and output it as encrypted.\+dat\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{./des -\/e -\/i plaintext.txt -\/o encrypted.dat -\/p your\_password}

\end{DoxyCode}
\hypertarget{md_README_autotoc_md10}{}\doxysubsection{Decrypt a File}\label{md_README_autotoc_md10}
To decrypt the encrypted.\+dat file back to decrypted.\+txt\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{./des -\/d -\/i encrypted.dat -\/o decrypted.txt -\/p your\_password}

\end{DoxyCode}
